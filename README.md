# AutoPIPE-DBConverter
AutoPIPE Results Databases are by default exported in the SQLite3 database file format (.DB). This file format requires SQLite3 compatible software to either read the database, or to convert the database tables into a more universal format for the end user.

This project provides a GUI based application to extract the Support and Restraint tables from an AutoPIPE Results Database, exporting these as .CSV files to a selected location. This makes it easier to use the Support and Anchor loads from an AutoPIPE analysis as inputs for Engineering design and analysis of support structures and support furniture.

# How It Works
This project was coded in Python, using standard libraries.

# Installation and Use
This repository contains both a pre-compiled executable, which can be downloaded and used immediately.

Alternatively, the .py file, containing the code can be downloaded and compiled by the end user.
